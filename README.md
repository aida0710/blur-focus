# Blur Focus

ホバーしているテキスト以外をぼかすChrome拡張機能。
ドキュメントを読むときに文字が滑ってしまい集中できない問題を解決します。

## 主な機能

- リアルタイムブラー効果 - ページリロード不要で即座に反映
- ブラー強度のカスタマイズ - 2px～15pxの範囲で調整可能
- 対象要素の選択 - ブラーを適用する要素タイプを個別に選択
- サイトごとの設定 - 特定のサイトでのみ有効/無効を設定
- キーボードショートカット - `Ctrl+Shift+B` (Mac: `⌘+Shift+B`) で即座にトグル
- 動的要素対応 - SPAなど動的に生成される要素にも自動対応

## インストール方法

### 方法1: リリースページからダウンロード

1. [Releases](https://github.com/aida0710/blur-focus/releases)ページにアクセス
2. 最新バージョンの`blur-focus-vX.X.X.zip`をダウンロード
3. ZIPファイルを解凍
4. Chrome/Edgeで`chrome://extensions/`を開く
5. 「デベロッパーモード」を有効化
6. 「パッケージ化されていない拡張機能を読み込む」をクリック
7. 解凍したフォルダを選択

### 方法2: ソースからビルド

```bash
git clone https://github.com/aida0710/blur-focus.git
cd blur-focus
npm install
npm run build

# ブラウザで dist フォルダを読み込む
```

## 使い方

### 基本操作

1. 拡張機能アイコンをクリック - ポップアップが開きます
2. 「ブラーを有効にする」をクリック - 即座にブラーが適用されます
3. テキストにホバー - ホバーした部分だけがクリアに表示されます

### キーボードショートカット

- `Ctrl+Shift+B` (Mac: `⌘+Shift+B`) - ブラーのオン/オフ切り替え

### 詳細設定

「設定を開く」ボタンをクリックすると、以下の設定が可能です：

#### ブラー強度

スライダーでぼかしの強さを調整できます（2px～15px）。リアルタイムでプレビューを確認できます。

#### 対象要素

ブラーを適用する要素タイプを選択できます：
- `<h1>` ～ `<h6>` - 見出し
- `<p>` - 段落
- `<a>` - リンク
- `<span>` - インラインテキスト
- `<ul>`, `<li>` - リスト
- `<label>` - ラベル
- `<code>` - コード

#### サイトごとの設定

特定のサイトでのみブラーを有効/無効にできます：
1. URLパターンを入力（例: `example.com`）
2. 「追加」をクリック
3. チェックボックスで有効/無効を切り替え

「現在のサイト」ボタンで現在開いているサイトを簡単に追加できます。
